Como centralizar uma div?

Overflowia Modelo v2.0

Oi, estou fazendo meu projeto final. Como posso centralizar uma div perfeitamente na tela usando CSS?

Olá! Que legal o seu PFC. Centralizar uma div é um clássico da programação web. A forma mais moderna e eficiente hoje em dia é usar Flexbox. Veja o exemplo abaixo:

CSS (Flexbox)
.container { display: flex; justify-content: center; align-items: center; height: 100vh; }
