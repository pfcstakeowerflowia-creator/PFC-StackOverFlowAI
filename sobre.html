Acessibilidade e Engenharia de Software

Este site foi construído seguindo as diretrizes do WCAG para garantir que leitores de tela como NVDA e JAWS funcionem perfeitamente.

  • HTML Semântico: Uso de tags <main>, <nav> e <article> para navegação lógica.
  • Atributos ARIA: Labels descritivos para elementos interativos.
  • Alto Contraste: Paleta baseada em tons escuros (UI Dark Mode) com contraste de texto otimizado para baixa visão.
  • Tags Alt: Todas as imagens e ícones possuem descrições para cegos.
